# 프론트엔드
- 아래와 같이 디렉토리 구조를 잡음
```
├── app               -- 페이지 및 글로벌 파일 디렉토리
│   ├── favicon.ico   -- 패비콘  
│   ├── fonts         -- 폰트
│   ├── globals.css   -- 글로벌 css
│   ├── layout.tsx    -- 글로벌 layout
│   └── page.tsx      -- Main 페이지
└── component         -- 컴포넌트 디렉토리
```

- 아래와 같이 `탑 메뉴`, `사이드 컨텐츠 공간`, `메인 컨텐츠 공간`을 `Grid Layout` 으로 잡음
